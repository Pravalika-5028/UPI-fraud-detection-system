# UPI Fraud Detection using Machine Learning

Project Overview

This project builds a Machine Learning model to detect fraudulent UPI (Unified Payments Interface) transactions. The model analyzes transaction-related features and predicts whether a transaction is **Fraudulent** or **Legitimate**.

The objective is to help financial institutions identify suspicious transactions in real time and reduce financial losses.

##  Dataset

The dataset contains UPI transaction information such as:

- Transaction Amount
- Sender Details
- Receiver Details
- Transaction Time
- Payment Method
- Device Information
- Location
- Fraud Flag (Target Variable)

> **Target Variable**
>
> - `0` → Legitimate Transaction
> - `1` → Fraudulent Transaction
